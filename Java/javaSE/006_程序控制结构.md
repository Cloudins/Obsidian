## if-else 结构

```java
//单分支
if( 条件表达式 ){
		执行代码块;
}
//多分支
if( 条件表达式 ){
	执行代码块;
}else{
	执行代码块;
}
```

## 空心金字塔

```java
int level = 5;  
  
for (int j = 1; j <= level; j++) {  
    for (int i = 1; i <= level - j; i++) {  
        System.out.print(" ");  
    }  
  
    for (int k = 1; k <= 2 * j - 1; k++) {  
        if (k == 1 || k == 2 * j - 1 || j == 5 ){  
            System.out.print("*");  
        }else {  
            System.out.print(" ");  
        }  
    }    System.out.println("");  
}
```
